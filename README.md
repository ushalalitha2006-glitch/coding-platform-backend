## 🚀 Coding Platform Backend

A backend system for a coding platform built using Spring Boot.

## 📌 Features

- User Registration & Login
- Problem Management
- Code Submission API
- RESTful APIs
- H2 Database Integration

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## 📡 API Endpoints

### Users
- POST /api/users
- GET /api/users

### Problems
- POST /api/problems
- GET /api/problems

### Submissions
- POST /api/submissions

## ⚙️ Run Project

```bash
mvn spring-boot:run
